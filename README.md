## 🎯 О проекте

Проект по обучению нейросети для автоматического распознавания **марки**, **модели** и **года выпуска** автомобилей с изображений городских камер видеонаблюдения. Применятся архитектура YOLO для детекции и классификации объектов в реальном времени.

### Основные возможности

- 🔍 **Распознавание марки автомобиля** (Toyota, BMW, Mercedes, Audi и др.)
- 🚙 **Определение модели** (конкретная модель производителя)
- 📅 **Классификация года выпуска** (по поколениям/годам)
- 🎥 **Работа с городских камер** (оптимизация для уличного видеонаблюдения)

### Применение

| Область | Применение |
|---------|------------|
| **Городское видеонаблюдение** | Автоматический учёт транспорта, мониторинг трафика |
| **Парковки и платные зоны** | Контроль доступа, автоматическая оплата |
| **Безопасность** | Поиск угнанных автомобилей, идентификация |
| **Трафик-аналитика** | Статистика по маркам, возрастному составу автопарка |
| **Smart City** | Интеграция в умные города, управление трафиком |


## 📊 Датасет

### Описание

Датасет состоит из изображений автомобилей, полученных с **городских камер видеонаблюдения**:

- **Источник**: Городские камеры видеонаблюдения
- **Тип данных**: Изображения автомобилей с разных ракурсов
- **Условия**: Разное освещение, погодные условия, углы съёмки
- **Разметка**: Вручную размеченные bounding box с классами

### Формат данных

| Параметр | Значение |
|----------|----------|
| Формат разметки | YOLO (txt файлы) |
| Bounding box | X_center, Y_center, Width, Height |
| Нормализация | 0-1 (относительно размера изображения) |
| Классы | Множественные (марка + модель + год) |

## 🤖 Модель

### Архитектура

- **Нейросеть**: YOLO (You Only Look Once)
- **Задача**: Детекция объектов + классификация
- **Фреймворк**: PyTorch
- **Оптимизация**: Для городских камер и уличных условий

### Особенности

- ✅ Детекция в реальном времени
- ✅ Работа при разном освещении
- ✅ Устойчивость к погодным условиям
- ✅ Распознавание с разных углов
- ✅ Высокая точность локализации
